# 财务考勤综合系统 - 功能说明及使用方法

## 一、系统简介

本系统是一款专为室内装修行业设计的财务管理与考勤管理移动应用，适用于Android 8.0（API 26）及以上版本的手机。系统集成了工程收支记录、多项费用管理、多时段考勤打卡、灵活工资发放、工作内容记录和工程项目管理等功能，帮助装修从业者实现数字化管理。

## 二、核心功能

### 1. 财务管理

#### 1.1 工程收支记录
- **功能说明**：记录装修工程项目的收入和支出，包括合同款项、材料费、人工费等
- **使用方法**：
  1. 点击底部导航"财务"标签
  2. 点击"添加记录"按钮
  3. 选择记录类型（收入/支出）
  4. 输入金额、项目名称、日期
  5. 可选填写备注信息
  6. 点击"保存"完成记录

#### 1.2 交通费用管理
- **功能说明**：记录因工作产生的交通相关费用，包括油费、过路费、停车费等
- **使用方法**：
  1. 进入"财务"页面
  2. 点击"添加记录"，选择"交通费用"类型
  3. 输入费用金额、费用类型（油费/过路费/停车费等）
  4. 选择日期，可选关联项目
  5. 点击"保存"

#### 1.3 办公用品管理
- **功能说明**：记录工作中使用的办公用品采购费用
- **使用方法**：
  1. 进入"财务"页面
  2. 点击"添加记录"，选择"办公用品"类型
  3. 输入物品名称、数量、单价
  4. 系统自动计算总价
  5. 选择日期，点击"保存"

#### 1.4 生活食材管理
- **功能说明**：记录工作期间的生活开支，包括餐饮、食材采购等
- **使用方法**：
  1. 进入"财务"页面
  2. 点击"添加记录"，选择"生活食材"类型
  3. 输入食材名称、金额
  4. 选择日期，点击"保存"

#### 1.5 财务查询与导出
- **功能说明**：按日期、类型、项目筛选财务记录，支持导出为Excel
- **使用方法**：
  1. 进入"财务"页面
  2. 使用筛选条件（日期范围、记录类型、项目）
  3. 点击"查询"查看结果
  4. 点击"导出"生成Excel文件

### 2. 考勤管理

#### 2.1 多时段打卡
- **功能说明**：支持上午、下午、加班及自定义时间打卡
- **使用方法**：
  1. 点击底部导航"考勤"标签
  2. 选择要打卡的人员
  3. 选择打卡时段（上午/下午/加班/自定义）
  4. 可选关联项目
  5. 点击"上班打卡"记录开始时间
  6. 工作结束后点击"下班打卡"记录结束时间
  7. 系统自动计算该时段工作时长

#### 2.2 自定义时间打卡
- **功能说明**：支持用户自定义时间段进行打卡
- **使用方法**：
  1. 进入"考勤"页面
  2. 选择"自定义时间"时段
  3. 输入开始时间和结束时间
  4. 系统自动计算时长并保存

#### 2.3 考勤记录查询
- **功能说明**：按日期范围、人员、项目查询考勤记录
- **使用方法**：
  1. 进入"考勤"页面
  2. 设置筛选条件（日期范围、人员、项目）
  3. 点击"查询"查看考勤统计

### 3. 工资管理

#### 3.1 工资发放记录
- **功能说明**：记录工资发放，支持灵活的日工资标准
- **使用方法**：
  1. 点击底部导航"工资"标签
  2. 点击"添加工资记录"
  3. 选择人员
  4. 输入发放日期
  5. 输入工作天数
  6. 输入日工资标准（可修改，临时工必填）
  7. 系统自动计算：应发工资 = 工作天数 × 日工资标准
  8. 可选输入实发工资（默认等于应发工资）
  9. 可选填写备注
  10. 点击"保存"

#### 3.2 人员类型说明
- **固定员工**：可以设置参考日工资标准，发放工资时会自动填充参考标准，可手动修改
- **临时工**：每次发放工资时都需要输入日工资标准，无固定参考值

#### 3.3 工资查询
- **功能说明**：按人员、日期范围查询工资发放记录
- **使用方法**：
  1. 进入"工资"页面
  2. 设置筛选条件（人员、日期范围）
  3. 点击"查询"查看工资汇总

### 4. 工作内容记录

#### 4.1 记录工作内容
- **功能说明**：记录当日完成的工作任务和内容摘要
- **使用方法**：
  1. 点击底部导航"工作记录"标签
  2. 点击"添加工作记录"
  3. 选择人员
  4. 输入工作日期
  5. 输入工作内容描述
  6. 可选关联项目
  7. 点击"保存"

#### 4.2 工作记录查询
- **功能说明**：按人员查询工作记录
- **使用方法**：
  1. 进入"工作记录"页面
  2. 选择要查询的人员
  3. 点击"查询"查看记录

### 5. 人员管理

#### 5.1 添加人员
- **功能说明**：添加新的施工人员
- **使用方法**：
  1. 点击底部导航"人员"标签
  2. 点击"添加人员"
  3. 输入姓名
  4. 选择人员类型（固定员工/临时工）
  5. 如果是固定员工，可输入参考日工资标准
  6. 可选输入电话、入职日期
  7. 点击"保存"

#### 5.2 编辑/删除人员
- **功能说明**：修改人员信息或删除人员
- **使用方法**：
  1. 进入"人员"页面
  2. 点击人员卡片进入详情
  3. 点击"编辑"修改信息
  4. 点击"删除"移除人员

### 6. 工程项目管理

#### 6.1 创建项目
- **功能说明**：创建新的工程项目
- **使用方法**：
  1. 点击底部导航"项目"标签
  2. 点击"添加项目"
  3. 输入项目名称
  4. 输入开始日期
  5. 可选输入结束日期、项目地址、客户姓名、联系电话
  6. 可选输入合同金额
  7. 选择项目状态（进行中/已完成/已暂停）
  8. 可选填写备注
  9. 点击"保存"

#### 6.2 项目统计

##### 6.2.1 人员工时统计
- **功能说明**：自动统计项目中每个人员的工作天数和总工时
- **使用方法**：
  1. 进入"项目"页面
  2. 点击项目卡片进入详情
  3. 查看"人员工时"统计表
  4. 显示每个人员的工作天数和总工时

##### 6.2.2 项目费用汇总
- **功能说明**：自动汇总项目的所有费用
- **使用方法**：
  1. 进入"项目"页面
  2. 点击项目卡片进入详情
  3. 查看"费用汇总"统计
  4. 显示：
     - 材料费（工程收支）
     - 运输费（交通费用）
     - 办公用品费
     - 生活食材费
     - 总费用

## 三、使用技巧

### 1. 考勤与项目关联
- 建议在打卡时关联项目，这样可以方便地统计每个项目的人员投入
- 项目统计会自动汇总关联到该项目的所有考勤记录

### 2. 财务记录与项目关联
- 工程收支、交通费用等财务记录可以关联到项目
- 项目费用汇总会自动计算关联到该项目的所有费用

### 3. 工资发放建议
- 对于固定员工，先设置参考日工资标准，发放时系统会自动填充
- 对于临时工，每次发放时根据实际情况输入日工资标准
- 建议定期（如每月）发放一次工资，便于管理

### 4. 数据导出
- 定期导出财务、考勤、工资数据作为备份
- 导出的Excel文件可以用Excel或WPS打开查看

## 四、注意事项

1. **数据备份**：建议定期导出数据进行备份
2. **打卡规范**：按时打卡，确保考勤记录准确
3. **项目关联**：尽量将考勤和财务记录关联到项目，便于统计分析
4. **工资发放**：仔细核对工作天数和日工资标准后再保存
5. **系统兼容性**：本系统仅支持Android 8.0（API 26）及以上版本

## 五、常见问题

### Q1：如何修改已保存的记录？
A：点击对应记录进入详情页，点击"编辑"按钮进行修改。

### Q2：如何删除记录？
A：点击对应记录进入详情页，点击"删除"按钮。注意：删除后无法恢复。

### Q3：工资标准可以随时修改吗？
A：可以。每次发放工资时都可以修改日工资标准，系统会记录每次发放时使用的标准。

### Q4：临时工和固定员工有什么区别？
A：
- 固定员工：有参考日工资标准，发放时自动填充，可修改
- 临时工：无固定标准，每次发放时都需要输入日工资标准

### Q5：项目统计是如何计算的？
A：
- 人员工时：汇总所有关联到该项目的考勤记录
- 费用汇总：汇总所有关联到该项目的财务记录，按类型分类
